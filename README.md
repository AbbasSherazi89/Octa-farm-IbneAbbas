# Octa Farm

A comprehensive farming analytics dashboard built with React, featuring interactive data visualizations, FAQ components, and smooth scrolling navigation.

**Live URL:** https://ibneabbas-soft-taiyaki-d352f.netlify.app/
## 🚀 Tech Stack

- **React 18** - UI library
- **React Router DOM v6** - Routing and navigation
- **Bootstrap 5 & React-Bootstrap** - Responsive UI components
- **SASS** - CSS preprocessor for advanced styling
- **React Google Charts** - Google Charts integration for data visualization
- **React Linechart** - Line chart components for analytics
- **React FAQ Component** - Collapsible FAQ sections
- **React Scroll** - Smooth scrolling animations
- **React Icons** - Icon library

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
